# Introduccion
Esta aplicacion la cree para practicar mis habilidades en desarrollo mobil para un ramo que tuve en el instituto.
Esta app esta conectada a una API , la api de OmdB (https://www.omdbapi.com/). La finalidad de esta app es ser un buscador de peliuclas
# Elementos importantes
Esta app ocupa Retrofit para hacer la conexion con la api y ocupa dependencias como glide y binder para mostrar elementos desde el Json

## Elementos de la rama main
Esta rama contiene la primera version de la app, la cual solo contiene dos actividade, una para buscar una pelicula y la otra para ver la informacion de la pelicula
